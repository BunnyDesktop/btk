btk_spin_button_new_with_range ()

BtkWidget *
btk_spin_button_new_with_range (gdouble min,
                                gdouble max,
                                gdouble step);

This is a convenience constructor that allows creation of a numeric BtkSpinButton without manually creating an adjustment. The value is initially set to the minimum value and a page increment of 10 * step is the default. The precision of the spin button is equivalent to the precision of step .

Note that the way in which the precision is derived works best if step is a power of ten. If the resulting precision is not suitable for your needs, use btk_spin_button_set_digits() to correct it.

Parameters

/* show leading zeros */
static gboolean
on_output (BtkSpinButton *spin,
           gpointer       data)
{
   BtkAdjustment *adj;
   gchar *text;
   int value;

   adj = btk_spin_button_get_adjustment (spin);
   value = (int)btk_adjustment_get_value (adj);
   text = g_strdup_printf ("%02d", value);
   btk_entry_set_text (BTK_ENTRY (spin), text);
   g_free (text);

   return TRUE;
}
