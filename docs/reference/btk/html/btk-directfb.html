Using BTK+ on DirectFB

The DirectFB port of BTK+ is an implementation of BDK (and therefore BTK+) on top of the DirectFB libraries.

Like the obsolete linux-fb port of BTK+, it runs on the Linux framebuffer, but the DirectFB libraries provide additional features like hardware graphics acceleration, input device handling and abstraction and an integrated windowing system.

Build requirements

Beyond the usual BTK+ build requirements, the DirectFB backend (obviously) needs the DirectFB libraries (at least 0.9.21) and bairo compiled with DirectFB support.
