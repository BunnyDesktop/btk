Description

The BtkBin widget is a container with just one child. It is not very useful itself, but it is useful for deriving subclasses, since it provides common code needed for handling a single child widget.

Many BTK+ widgets are subclasses of BtkBin, including BtkWindow, BtkButton, BtkFrame, BtkHandleBox or BtkScrolledWindow.

BtkWidget *
btk_bin_get_child (BtkBin *bin);

Gets the child of the BtkBin, or NULL if the bin contains no child widget. The returned widget does not have a reference added, so you do not need to unref it.
