Implemented Interfaces

BtkToolItemGroup implements BatkImplementorIface, BtkBuildable and BtkToolShell.

Includes

#include <btk/btk.h>

Description

A BtkToolItemGroup is used together with BtkToolPalette to add BtkToolItems to a palette like container with different categories and drag and drop support.
