Description

A BtkSpinner widget displays an icon-size spinning animation. It is often used as an alternative to a BtkProgressBar for displaying indefinite activity, instead of actual progress.

To start the animation, use btk_spinner_start(), to stop it use btk_spinner_stop().

The “cycle-duration” style property

The duration in milliseconds for the spinner to complete a full cycle.

Flags: Read

Allowed values: >= 500

Default value: 1000

Since: 2.20

The “num-steps” style property

  “num-steps”                buint

The number of steps for the spinner to complete a full loop. The animation will complete a full cycle in one second by default (see the “cycle-duration” style property).

Owner: BtkSpinner

Flags: Read

Allowed values: >= 1

Default value: 12
