btk_adjustment_set_lower ()

void
btk_adjustment_set_lower (BtkAdjustment *adjustment,
                          bdouble lower);

Sets the minimum value of the adjustment.

When setting multiple adjustment properties via their individual setters, multiple "changed" signals will be emitted. However, since the emission of the "changed" signal is tied to the emission of the "BObject::notify" signals of the changed properties, it's possible to compress the "changed" signals into one by calling g_object_freeze_notify() and g_object_thaw_notify() around the calls to the individual setters.

Alternatively, using a single g_object_set() for all the properties to change, or using btk_adjustment_configure() has the same effect of compressing "changed" emissions.
