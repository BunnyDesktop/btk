Use btk_combo_box_new_with_model_and_entry() instead

Creates a new BtkComboBoxEntry which has a BtkEntry as child and a list of strings as popup. You can get the BtkEntry from a BtkComboBoxEntry using BTK_ENTRY (BTK_BIN (combo_box_entry)->child). To add and remove strings from the list, just modify model using its data manipulation API.

btk_combo_box_entry_new_text ()

BtkWidget *
btk_combo_box_entry_new_text (void);

btk_combo_box_entry_new_text is deprecated and should not be used in newly-written code.

Convenience function which constructs a new editable text combo box, which is a BtkComboBoxEntry just displaying strings. If you use this function to create a text combo box, you should only manipulate its data source with the following convenience functions: btk_combo_box_append_text(), btk_combo_box_insert_text(), btk_combo_box_prepend_text() and btk_combo_box_remove_text().

Returns

A new text BtkComboBoxEntry.

Since: 2.4
