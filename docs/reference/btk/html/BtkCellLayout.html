btk_cell_layout_pack_start ()

void
btk_cell_layout_pack_start (BtkCellLayout *cell_layout,
                            BtkCellRenderer *cell,
                            bboolean expand);

Packs the cell into the beginning of cell_layout . If expand is FALSE, then the cell is allocated no more space than it needs. Any unused space is divided evenly between cells for which expand is TRUE.

Note that reusing the same cell renderer is not supported.

Parameters

A BtkCellLayout.

btk_cell_layout_add_attribute ()

void
btk_cell_layout_add_attribute (BtkCellLayout *cell_layout,
                               BtkCellRenderer *cell,
                               const bchar *attribute,
                               bint column);

Adds an attribute mapping to the list in cell_layout . The column is the column of the model to get a value from, and the attribute is the parameter on cell to be set from the value. So for example if column 2 of the model contains strings, you could have the "text" attribute of a BtkCellRendererText get its values from column 2.

Parameters

cell_layout

A BtkCellLayout.

  

cell

A BtkCellRenderer.

  

attribute

An attribute on the renderer.

  

column

The column position on the model to get the attribute from.

Sets the BtkCellLayoutDataFunc to use for cell_layout . This function is used instead of the standard attributes mapping for setting the column value, and should set the value of cell_layout 's cell renderer(s) as appropriate. func may be NULL to remove and older one.
