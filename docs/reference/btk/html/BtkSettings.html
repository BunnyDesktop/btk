typedef struct _BtkSettings BtkSettings;

struct BtkSettingsValue

struct BtkSettingsValue {
  /* origin should be something like "filename:linenumber" for rc files,
   * or e.g. "XProperty" for other sources
   */
  bchar *origin;

  /* valid types are LONG, DOUBLE and STRING corresponding to the token parsed,
   * or a GSTRING holding an unparsed statement
   */
  BValue value;
};

The “btk-color-scheme” property

  “btk-color-scheme”         char *

A palette of named colors for use in themes. The format of the string is

name1: color1
name2: color2
...

Color names must be acceptable as identifiers in the

btkrc syntax, and

color specifications must be in the format accepted by bdk_color_parse().

Note that due to the way the color tables from different sources are merged, color specifications will be converted to hexadecimal form when getting this property.

Starting with BTK+ 2.12, the entries can alternatively be separated by ';' instead of newlines:

name1: color1; name2: color2; ...

Owner: BtkSettings

Flags: Read / Write

Default value: ""

Since: 2.10

The “btk-tooltip-browse-timeout” property

  “btk-tooltip-browse-timeout” int

Controls the time after which tooltips will appear when browse mode is enabled, in milliseconds.

Browse mode is enabled when the mouse pointer moves off an object where a tooltip was currently being displayed. If the mouse pointer hits another object before the browse mode timeout expires (see “btk-tooltip-browse-mode-timeout”), it will take the amount of milliseconds specified by this setting to popup the tooltip for the new object.

Owner: BtkSettings

Flags: Read / Write

Allowed values: >= 0

Default value: 60

Since: 2.12
