Property Details

The “xalign” property

  “xalign”                   float

The horizontal alignment, from 0 (left) to 1 (right). Reversed for RTL layouts.

Owner: BtkMisc

Flags: Read / Write

Allowed values: [0,1]

Default value: 0.5

  “xpad”                     int

The amount of space to add on the left and right of the widget, in pixels.

Owner: BtkMisc

Flags: Read / Write

Allowed values: >= 0

Default value: 0
