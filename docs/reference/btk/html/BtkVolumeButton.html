Description

BtkVolumeButton is a subclass of BtkScaleButton that has been tailored for use as a volume control widget with suitable icons, tooltips and accessible labels.

btk_volume_button_new (void);

Creates a BtkVolumeButton, with a range between 0.0 and 1.0, with a stepping of 0.02. Volume values can be obtained and modified using the functions from BtkScaleButton.
