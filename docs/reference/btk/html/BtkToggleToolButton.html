btk_toggle_tool_button_new_from_stock ()

BtkToolItem *
btk_toggle_tool_button_new_from_stock (const gchar *stock_id);

Creates a new BtkToggleToolButton containing the image and text from a stock item. Some stock ids have preprocessor macros like BTK_STOCK_OK and BTK_STOCK_APPLY.

It is an error if stock_id is not a name of a stock item.

btk_toggle_tool_button_set_active ()

void
btk_toggle_tool_button_set_active (BtkToggleToolButton *button,
                                   gboolean is_active);

Sets the status of the toggle tool button. Set to TRUE if you want the BtkToggleButton to be 'pressed in', and FALSE to raise it. This action causes the toggled signal to be emitted.

Parameters

button

a BtkToggleToolButton

  

is_active

whether button should be active

btk_toggle_tool_button_get_active (BtkToggleToolButton *button);
