Prerequisites

BtkToolShell requires BtkWidget.

Known Implementations

BtkToolShell is implemented by BtkToolItemGroup and BtkToolbar.

Includes

#include <btk/btk.h>

Description

The BtkToolShell interface allows container widgets to provide additional information when embedding BtkToolItem widgets.

see_also : BtkToolbar, BtkToolItem

btk_tool_shell_rebuild_menu ()

void
btk_tool_shell_rebuild_menu (BtkToolShell *shell);

Calling this function signals the tool shell that the overflow menu item for tool items have changed. If there is an overflow menu and if it is visible when this function it called, the menu will be rebuilt.

Tool items must not call this function directly, but rely on btk_tool_item_rebuild_menu() instead.
