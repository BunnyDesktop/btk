btk_print_context_set_bairo_context ()

void
btk_print_context_set_bairo_context (BtkPrintContext *context,
                                     bairo_t *cr,
                                     double dpi_x,
                                     double dpi_y);

Sets a new bairo context on a print context.

This function is intended to be used when implementing an internal print preview, it is not needed for printing, since BTK+ itself creates a suitable bairo context in that case.

Parameters

context

a BtkPrintContext

  

cr

the bairo context

  

dpi_x

the horizontal resolution to use with cr

 

dpi_y

the vertical resolution to use with cr
