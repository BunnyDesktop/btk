btk_tree_selection_set_select_function ()

void
btk_tree_selection_set_select_function
                               (BtkTreeSelection *selection,
                                BtkTreeSelectionFunc func,
                                gpointer data,
                                GDestroyNotify destroy);

Sets the selection function. If set, this function is called before any node is selected or unselected, giving some control over which nodes are selected. The select function should return TRUE if the state of the node may be toggled, and FALSE if the state of the node should be left unchanged.

btk_tree_selection_get_selected ()

gboolean
btk_tree_selection_get_selected (BtkTreeSelection *selection,
                                 BtkTreeModel **model,
                                 BtkTreeIter *iter);

Sets iter to the currently selected node if selection is set to BTK_SELECTION_SINGLE or BTK_SELECTION_BROWSE. iter may be NULL if you just want to test if selection has any selected nodes. model is filled with the current model as a convenience. This function will not work if you use selection is BTK_SELECTION_MULTIPLE.

Parameters

selection

A BtkTreeSelection.

  

model

A pointer to set to the BtkTreeModel, or NULL.

[out][allow-none][transfer none]

iter

The BtkTreeIter, or NULL.

[out][allow-none]

Returns

TRUE, if there is a selected node.

btk_tree_selection_get_selected_rows ()

GList *
btk_tree_selection_get_selected_rows (BtkTreeSelection *selection,
                                      BtkTreeModel **model);

Creates a list of path of all selected rows. Additionally, if you are planning on modifying the model after calling this function, you may want to convert the returned list into a list of BtkTreeRowReferences. To do this, you can use btk_tree_row_reference_new().

To free the return value, use:

1
2
 
g_list_foreach (list, (GFunc) btk_tree_path_free, NULL);
g_list_free (list);

Parameters

selection

A BtkTreeSelection.

  

model

A pointer to set to the BtkTreeModel, or NULL.

[out][allow-none][transfer none]

Returns

A GList containing a BtkTreePath for each selected row.

[element-type BtkTreePath][transfer full]
