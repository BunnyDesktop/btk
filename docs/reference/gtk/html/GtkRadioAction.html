btk_radio_action_get_group ()

GSList *
btk_radio_action_get_group (BtkRadioAction *action);

Returns the list representing the radio group for this object. Note that the returned list is only valid until the next change to the group.

A common way to set up a group of radio group is the following:

BtkRadioAction *action;
 
while (/* more actions to add */)
  {

     group = btk_radio_action_get_group (action);
  }

The “value” property

  “value”                    int

The value is an arbitrary integer which can be used as a convenient way to determine which action in the group is currently active in an ::activate or ::changed signal handler. See btk_radio_action_get_current_value() and BtkRadioActionEntry for convenient ways to get and set this property.

Owner: BtkRadioAction

Flags: Read / Write

Default value: 0

Since: 2.4

Signal Details

The “changed” signal

void
user_function (BtkRadioAction *action,
               BtkRadioAction *current,
               gpointer        user_data)

The ::changed signal is emitted on every member of a radio group when the active member is changed. The signal gets emitted after the ::activate signals for the previous and current active members.
