Description

The BtkAspectFrame is useful when you want pack a widget so that it can resize but always retains the same aspect ratio. For instance, one might be drawing a small preview of a larger image. BtkAspectFrame derives from BtkFrame, so it can draw a label and a frame around the child. The frame will be "shrink-wrapped" to the size of the child.
