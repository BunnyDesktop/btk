btk_cell_view_set_displayed_row ()

void
btk_cell_view_set_displayed_row (BtkCellView *cell_view,
                                 BtkTreePath *path);

Sets the row of the model that is currently displayed by the BtkCellView. If the path is unset, then the contents of the cellview "stick" at their last value; this is not normally a desired result, but may be a needed intermediate state if say, the model for the BtkCellView becomes temporarily empty.

Parameters

cell_view

a BtkCellView

  

path

a BtkTreePath or NULL to unset.

[allow-none]

Since: 2.6
